# HNL Monitor Stands

Product website for HNL Monitor Stands — a portable, lightweight monitor stand designed for students, freelancers, and anyone working on the go.

## Pages

- `/` — Home (hero carousel, product highlights)
- `/product` — Product details
- `/about_us` — About Us
- `/blog` — Blog

## Running Locally

```bash
pip install -r requirements.txt
flask --app happ run
```

Then open http://127.0.0.1:5000 in your browser.
